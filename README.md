# 3D Spine Portfolio Scene

Static Three.js scene with an iridescent biomechanical spine, project-card rail, colored dust clusters, and reference-inspired cinematic grading.

## Local Run

```bat
start-site.bat
```

Then open:

```text
http://127.0.0.1:5180/
```

The site is also deployable as static files through GitHub Pages because `index.html` loads local `vendor/` and `assets/` files directly.
